# Healthcare Data Security using Blockchain

## 📌 Project Description
This project ensures secure storage and transfer of healthcare data using encryption and blockchain technology.

## 🚀 Features
- AES Encryption for data security 🔐
- SQLite Database for storage 💾
- Blockchain for tamper-proof records 🔗
- Flask API for accessing data 🌐

## 🛠 Tech Stack
- Python
- Pandas
- Flask
- SQLite
- Cryptography (AES)

## ▶️ How to Run
1. Install dependencies:
   pip install pandas flask pycryptodome

2. Run main program:
   python main.py

3. Run API:
   python api.py
